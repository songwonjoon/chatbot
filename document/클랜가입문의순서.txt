클랜 가입문의 순서

1. 프로필 설정
아래 링크를 클릭해 클랜 마크 채널로 들어가 주세요.
클랜 마크 채널
채널에서 신입 마크를 다운로드한 후, 해당 마크로 디스코드 프로필 사진을 변경해 주세요.
서버 닉네임을 "별명/나이" 형식으로 바꿔 주세요. 예시: 코카/31
* 참고: 나이는 만 나이가 아닌 현재 나이로 기재해 주세요.
2. 자기소개 작성
다음 링크를 클릭하여 자기소개 채널로 들어가 주세요.
자기소개 채널
채널에서 위에 있는 자기소개 양식을 복사하고, 본인의 간단한 자기소개서를 작성해 주세요. * 민감한 정보는 포함되지 않으며, 가입 날짜 기록 및 홍보 경로를 알기 위함입니다.
절대 사적으로 사용하지 않습니다.
3. 서버 규칙 확인
아래 링크를 클릭해 Bliss 룰 채널로 이동해 주세요.
Bliss 룰 채널
서버 규칙 10가지를 읽어 주세요.
4. 완료 후 연락
모든 절차가 완료되면 확인이라고 입력해주세요
